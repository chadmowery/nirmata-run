**3. Create StabilitySystem** (`src/game/systems/stability.ts`):
Per D-08 (dual drain), D-09 (linear escalation), D-11 (degraded state HP damage):

```typescript
export interface StabilityConfig {
  initialStability: number;      // 100
  perFloorChunkBase: number;     // 5 — base stability lost on entering a new floor
  perFloorChunkScale: number;    // 0.5 — additional per-floor scaling (floor 10 loses 5 + 9*0.5 = 9.5)
  perTurnBleedBase: number;      // 0.5 — base stability lost per turn
  perTurnBleedScale: number;     // 0.1 — additional per-floor scaling
  degradedDamagePerTurn: number; // 2 — HP damage per turn when stability is 0
}

export const DEFAULT_STABILITY_CONFIG: StabilityConfig = {
  initialStability: 100,
  perFloorChunkBase: 5,
  perFloorChunkScale: 0.5,
  perTurnBleedBase: 0.5,
  perTurnBleedScale: 0.1,
  degradedDamagePerTurn: 2,
};
```

The system follows factory pattern: `export function createStabilitySystem(world, eventBus, config = DEFAULT_STABILITY_CONFIG)` returning `{ init, dispose, applyFloorDrain, applyTurnBleed, applyDegradedDamage }`.

- `applyFloorDrain(entityId, floorNumber)`:
  - Get Stability component (guard: `if (!stability) return`)
  - Calculate chunk: `config.perFloorChunkBase + (floorNumber - 1) * config.perFloorChunkScale`
  - Subtract from `stability.current`, clamp to 0
  - Emit `STABILITY_CHANGED` with `{ entityId, oldValue, newValue, reason: 'floor_entry' }`
  - If `stability.current === 0`, emit `STABILITY_ZERO` with `{ entityId }`

- `applyTurnBleed(entityId, floorNumber)`:
  - Get Stability component (guard)
  - Calculate bleed: `config.perTurnBleedBase + (floorNumber - 1) * config.perTurnBleedScale`
  - Subtract, clamp to 0
  - Emit `STABILITY_CHANGED` with reason `'turn_bleed'`
  - If transitions from > 0 to 0, emit `STABILITY_ZERO`

- `applyDegradedDamage(entityId)`:
  - Get both Stability and Health components (guard both)
  - If `stability.current > 0`, return (not degraded)
  - Deal `config.degradedDamagePerTurn` HP damage
  - Emit `DEGRADED_DAMAGE` event with `{ entityId, damage: config.degradedDamagePerTurn }`
  - If HP reaches 0, emit `ENTITY_DIED` with `{ entityId, killerId: entityId }` (self-inflicted)

- `init()`:
  - Listen for `FLOOR_TRANSITION` → call `applyFloorDrain(playerId, payload.floorNumber)`
  - Listen for `TURN_END` (or equivalent post-turn event) → for player entity, call `applyTurnBleed` then `applyDegradedDamage`
