    - Reads `playerProfile`, `selectedShellIndex`, `draggedItem`, `dragOverSlot` from store
    - Gets Shell templates from localStorage cache or API (fetched in ShellTab)
    - Determines active Shell from selectedShellIndex
    - Computes equipped items for the active Shell by filtering `playerProfile.installedItems` where `shellId` matches
    - Computes available Vault items (unequipped) from `playerProfile.vault`

    - Unequip: drag from filled Shell slot to Stash list area — calls POST `/api/economy/uninstall` then moves item back to vault
    - Filter state: `activeFilter` useState ('all' | 'firmware' | 'augment' | 'software')
