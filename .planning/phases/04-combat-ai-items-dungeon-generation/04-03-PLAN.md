<task type="auto" tdd="true">
  <name>Task 2: Item pickup system</name>
  <files>src/game/systems/item-pickup.ts, src/game/systems/item-pickup.test.ts</files>
  <behavior>
    - System subscribes to ENTITY_MOVED events
    - On ENTITY_MOVED (only for player entities): check grid item layer at destination
    - For each item at destination: apply effect, emit ITEM_PICKED_UP, remove from grid, destroy entity
    - Heal effect: increase health.current by value, capped at health.max
    - Non-item entities at destination are ignored
    - Non-player movers don't trigger pickup
  </behavior>
  <action>
    1. Create src/game/systems/item-pickup.ts:
       ```typescript
       export function createItemPickupSystem(
         world: World,
         grid: Grid,
         eventBus: EventBus<GameEvents>
       ): ItemPickupSystem
       ```
       - `init()` subscribes to ENTITY_MOVED
       - `dispose()` unsubscribes
       - Internal `onEntityMoved(payload)`:
         a. Check if mover is the player (has Actor with isPlayer === true)
         b. If not player, return
         c. Get items at (toX, toY): `grid.getItemsAt(toX, toY)`
         d. For each itemId in items:
            - Check if entity has Item component (skip non-items)
            - Apply pickup effect if PickupEffect component exists:
              - 'heal': get player Health, increase current by value, cap at max, update component
            - Emit ITEM_PICKED_UP: `{ entityId: playerId, itemId }`
            - Remove from grid: `grid.removeItem(itemId, toX, toY)`
            - Destroy entity: `world.destroyEntity(itemId)`

    2. Create src/game/systems/item-pickup.test.ts:
       - Setup: World, Grid, EventBus. Player at (3, 3) with Health { current: 10, max: 20 }
       
       - **Pickup tests:**
         - Player moves to tile with health potion → item picked up
         - ITEM_PICKED_UP event emitted with correct entityId and itemId
         - Item removed from grid (getItemsAt returns empty)
         - Item entity destroyed (entityExists returns false)
       
       - **Heal effect tests:**
         - Health potion heals player: 10 + 5 = 15
         - Healing capped at max: current 18 + 5 = 20 (not 23)
         - Full health player still picks up item (heals to max, not above)
       
       - **Edge cases:**
         - Non-player entity moves to item tile → no pickup
         - Tile with no items → nothing happens
         - Multiple items on same tile → all picked up
         - Entity without Item component at position → ignored
  </action>
