<task type="auto" tdd="true">
  <name>Task 2: Combat resolution system</name>
  <files>src/game/systems/combat.ts, src/game/systems/combat.test.ts</files>
  <behavior>
    - Combat system subscribes to BUMP_ATTACK event from the event bus
    - On BUMP_ATTACK: resolve damage as attacker.attack.power − defender.defense.armor (minimum 1)
    - Apply damage to defender's health (current -= damage, clamped to 0)
    - Emit DAMAGE_DEALT event with { attackerId, defenderId, amount }
    - If defender health reaches 0: perform death sequence
    - Death sequence: remove entity from grid, roll loot table, spawn loot items, emit ENTITY_DIED, destroy entity
    - Loot items placed at the dead entity's position
    - If no Attack component on attacker or no Health on defender, skip silently
  </behavior>
  <action>
    1. Create src/game/systems/combat.ts:
       ```typescript
       export function createCombatSystem(
         world: World,
         grid: Grid,
         eventBus: EventBus<GameEvents>,
         entityFactory: EntityFactory
       ): CombatSystem
       ```
       - `init()` method subscribes to 'BUMP_ATTACK' on eventBus
       - `dispose()` method unsubscribes to prevent leaks
       - Internal `resolveBumpAttack(payload)`:
         a. Get attacker's Attack component, defender's Health and Defense components
         b. If attacker lacks Attack or defender lacks Health, return early
         c. Calculate damage: `Math.max(1, attacker.power - (defender?.armor ?? 0))`
         d. Apply: `defender.health.current = Math.max(0, defender.health.current - damage)`
         e. Update health component: `world.addComponent(defenderId, Health, updatedData)`
         f. Emit DAMAGE_DEALT: `{ attackerId, defenderId, amount: damage }`
         g. If health.current <= 0: call `handleDeath(defenderId, attackerId)`
       - Internal `handleDeath(entityId, killerId)`:
         a. Get Position of dead entity
         b. Remove from grid: `grid.removeEntity(entityId, pos.x, pos.y)`
         c. Roll loot table if LootTable component exists:
            - For each drop in lootTable.drops: `if (Math.random() < drop.chance)` →
              create item entity via factory, set its Position to dead entity's position, add to grid item layer
         d. Emit ENTITY_DIED: `{ entityId, killerId }`
         e. `world.destroyEntity(entityId)`

    2. Create src/game/systems/combat.test.ts:
       - Setup: create World, Grid, EventBus, mock EntityFactory. Create attacker (Attack { power: 5 }) and defender (Health { current: 10, max: 10 }, Defense { armor: 2 }) entities.

         - Loot chance roll fails (random > chance) → no item spawned
